This is where to find the software developed by the Apache Cocoon Project.

The latest release of Apache Cocoon is 2.1.10. Most parts, especially the core can be considered as very stable. Some parts exist that must be considered as alpha, especially those blocks which are marked as such. Previous releases of Apache Cocoon 1 and 2 can be found in the archive at http://archive.apache.org/dist/cocoon/.

Important Notes:

  • Download from your nearest mirror site (which also has useful instructions).
  • Verify PGP/GPG Signatures.
