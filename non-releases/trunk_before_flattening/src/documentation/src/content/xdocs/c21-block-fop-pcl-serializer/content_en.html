PCL Serializer

The pcl serializer takes fo xml events as input. By using the FOP project it creates pcl out of the sax events.

This serializer is optional and requires the fop package in the lib directory when building cocoon 2. However, the distribution includes this package already.

  • Name : fo2pcl
  • Class: org.apache.cocoon.serialization.PCLSerializer
  • Cacheable: yes.
