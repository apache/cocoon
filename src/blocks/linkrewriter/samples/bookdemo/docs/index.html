LinkRewriter demo

/index.html

This subsitemap demonstrates the LinkRewriterTransformer, which rewrites links with the aid of configurable InputModules

This particular demo is set up so that links of the form book:XYZ are resolved by looking in book.xml for the @label XYZ, and then using the associated href, with ..'s added as appropriate. This is done with an XMLFileModule and SimpleMappingMetaModule, defined in cocoon.xconf.
