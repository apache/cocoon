Third-Party Tutorials

  • cocooncenter.org has a few tutorials and articles
  • Galatea FlashGuides(TM)
  • Cocoon 2 How-To Pages
  • Leigh Dodds's Introduction to Cocoon 2 at the IBM developerWorks web site. Covers how to install and configure Cocoon, introduces the principles of the Cocoon 2 architecture and its key components, and provides example pipeline configurations which demonstrate how to construct a dynamic Web site using Cocoon and XSLT.
  • Leigh Dodds's Working with XML Server Pages in Apache Cocoon 2 at the IBM developerWorks web site. Introduces XML Server Pages (XSP), the Cocoon technology for generating dynamic XML content.
  • Leigh Dodds's Cocoon 2: Build database-driven sites at the IBM developerWorks web site. Introduces basic concepts of the Cocoon 2 architecture and XML Server Pages (XSP) as a means for creating and publishing dynamic XML content.
